# UI Decisions - Quina

- Brand: Quina, wordmark lowercase, square dot for the "i" implied via accent square in UI.
- Voice: direct and gentle, infinitive verbs in CTAs.
- Palette: sage tokens in OKLCH, semantic names in `src/styles.css`.
- Typography: Space Grotesk for headings, DM Sans for body; no italics.
- Handmade level: subtle via odd radius (10px), thin borders, soft single shadow.
- Column signature: 2px sage line at top of each column, varying by column order.
- Cards: hover lift, soft shadow; check button added like Trello.
- Empty states: simple text placeholder; SVG leaf planned for future.
- Motion: 150ms ease-out transitions; no cascades.
